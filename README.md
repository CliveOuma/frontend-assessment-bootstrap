# Frontend Assessment at Fourfront mgt - Digital Finance

This project recreates the provided UI design using HTML, CSS, JavaScript, and Bootstrap 5.


## Technologies Used

- HTML5
- CSS3
- Bootstrap 5.3
- JavaScript

## Features Implemented

- Responsive layout (mobile, tablet, desktop)
- Bootstrap grid system
- Collapsible membership sections
- Modal with form validation
- Toast notifications
- Floating WhatsApp action button
- Clean and organized code structure

## Project Structure

/css
  styles.css
/js
  scripts.js
index.html
README.md


## Key Interactivity

- Menu toggle (hamburger navigation)
- Membership accordion toggle
- Form validation with visual feedback
- Dynamic success state
